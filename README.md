# Food Management Admin Dashboard

A web-based admin dashboard that allows administrators to manage food items, users, and comments. This system is designed for simple and efficient CRUD operations with real-time updates using Firebase.

## Features

- User authentication (Register / Login)
- Protected routing based on authentication
- Add, update, and delete food items
- Category management
- View registered users
- Comment system for each food item
- Real-time updates using Firestore
- Dashboard overview
